# dotfiles
Collection of my customized dot files.
## Awesome
### Enivronment Variables
Add this lines to your `.bash_profile`.
```bash
export COINMARKETCAP_API="[https://pro.coinmarketcap.com]"
export OPENWEATHERMAP_API="[https://openweathermap.org]"
```
